This project focuses on analyzing a large sales dataset from Amazon to explore sales performance, customer behavior, and key business trends using data analysis and visualization technique ,

Project Objective
Analyze large sales data efficiently
Identify patterns and trends in sales and revenue
Understand customer behavior across regions
Visualize insights clearly for better decision-making

Dataset Description:

Large dataset with thousands of sales records

Each row represents a single transaction
Includes:
Product category
Price and discount
Quantity sold
Revenue
Customer rating
Region
Payment method
